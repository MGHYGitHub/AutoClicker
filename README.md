# AutoClicker 高级自动连点器

一个功能强大的自动化鼠标点击工具，支持多点位管理、独立点击设置、安全检测等高级功能。

## 🚀 功能特性

### 🎯 核心功能
- **多点位管理**：添加/删除/编辑/上下移动/重命名点位
- **独立点位设置**：每个点位可独立设置坐标、名称、点击键、延时、点击次数、点击间隔
- **坐标捕获**：支持全局 F2 快捷键捕获鼠标坐标
- **任务控制**：启动/暂停/继续/停止任务，支持安全线程化
- **随机化设置**：随机偏移(px ±) 与 随机延时(s ±)

### 🛡️ 安全特性
- **智能防误触**：检测到异常鼠标移动自动暂停任务
- **可调节安全阈值**：50-1000像素灵敏度设置
- **调试模式**：逐步执行，便于调试复杂任务
- **启动前确认**：显示任务摘要，避免误操作

### 🎨 界面体验
- **现代化UI**：采用玻璃态设计，支持浅色/深色主题切换
- **系统托盘**：支持最小化到系统托盘后台运行
- **实时预览**：实时显示鼠标坐标和任务进度
- **详细日志**：完整的操作日志和执行统计

### ⌨️ 快捷键支持
- **F2**：记录当前鼠标坐标（全局/窗口内）
- **F3**：开始/停止任务（全局）
- **F4**：暂停/继续任务（全局）

### 📊 统计报告
- **实时统计**：总尝试次数、成功/失败次数、循环进度
- **时间记录**：任务开始和结束时间
- **报告导出**：可导出为文本文件

### 🔧 高级功能
- **窗口管理**：自动窗口切换，确保点击在正确的应用窗口
- **键盘操作**：支持快捷键组合输入
- **配置管理**：导入/导出 JSON 格式配置
- **自动保存**：自动保存/加载上次配置

## 🎮 使用方法

### 基本操作流程
1. **启动程序**：运行 `autoclicker.py`
2. **获取坐标**：点击"开始获取坐标"，移动鼠标到目标位置按 F2
3. **配置任务**：设置循环次数、延时、随机化参数等
4. **启动任务**：点击"启动任务"或按 F3 开始执行

### 点位管理
- **添加点位**：在捕获模式下按 F2 或点击"添加当前坐标"
- **编辑点位**：双击列表中的点位进行详细设置
- **调整顺序**：使用"上移"/"下移"按钮调整执行顺序
- **删除点位**：选择点位后点击"删除选中"

### 任务设置
- **循环次数**：任务重复执行的次数
- **基础延时**：点位之间的基础等待时间
- **随机偏移**：±像素范围，使点击位置更自然
- **随机延时**：±秒范围，模拟人类操作间隔
- **启动倒计时**：任务开始前的准备时间

## 🛠️ 技术细节

### 架构设计
- **多线程安全**：任务在独立线程中运行，不影响UI响应
- **事件驱动**：使用 threading.Event 进行任务控制
- **模块化设计**：各功能模块独立，便于维护扩展

### 安全机制
- **鼠标移动检测**：实时监控鼠标移动，异常时自动暂停
- **坐标验证**：点击前验证坐标有效性
- **错误处理**：完善的异常捕获和处理
