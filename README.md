# AutoClicker 2.3 - 高级自动连点器

一个功能丰富的图形化自动点击工具，支持多点位管理、全局快捷键、托盘运行等高级功能。

## 功能特点

### 🎯 核心功能
- **多点位管理**：添加、删除、编辑、上下移动、重命名点击点位
- **独立点位设置**：每个点位可独立设置坐标、名称、点击键（左/右/中键）、延时
- **坐标捕获**：支持全局 F2 快捷键捕获鼠标当前位置
- **任务控制**：开始、暂停、继续、停止，支持安全线程化操作

### ⚙️ 高级设置
- **随机化设置**：
  - 随机偏移 (±px)：点击位置随机偏移
  - 随机延时 (±s)：点击间隔时间随机变化
- **安全机制**：检测到鼠标快速移动时自动暂停任务
- **启动选项**：
  - 任务预览确认
  - 启动倒计时
  - 调试模式（逐点位确认）

### 🎨 界面特性
- **主题切换**：支持浅色/深色主题
- **托盘支持**：最小化到系统托盘，托盘菜单控制
- **实时预览**：实时显示鼠标坐标和任务状态

### 📊 统计与日志
- **点击统计**：总点击数、成功/失败次数、耗时等
- **详细日志**：自动记录操作日志到 `autoclicker_log.txt`
- **执行报告**：可导出详细点击报告 `click_log.txt`

### 💾 数据管理
- **自动保存**：自动保存/加载上次配置 (`click_points.json`)
- **导入导出**：支持 JSON 格式配置导入导出
- **配置管理**：保存、另存为配置文件

## 系统要求

- **操作系统**：Windows (推荐) / Linux / macOS
- **Python**：3.6 或更高版本
- **屏幕分辨率**：支持各种分辨率

## 安装依赖

```bash
pip install pyautogui pystray Pillow keyboard requests
```

### 可选依赖说明
- `keyboard`：全局快捷键支持（Windows 可能需要管理员权限）
- `pystray + Pillow`：系统托盘支持
- `requests`：自动更新检查功能

## 使用方法

### 基本操作流程
1. **启动程序**：运行 `autoclicker.py`
2. **添加点位**：
   - 点击"开始获取坐标"或按 F2
   - 移动鼠标到目标位置按 F2 记录
   - 或使用"添加当前坐标"按钮
3. **配置参数**：设置点击次数、延时、循环次数等
4. **启动任务**：点击"启动任务"或按 F3

### 快捷键
- **F2**：记录当前鼠标坐标（全局/窗口内）
- **F3**：开始/停止任务（全局）
- **F4**：暂停/继续任务（全局）

### 点位管理
- **双击点位**：编辑坐标、名称、按键、延时
- **右键菜单**：删除、重命名、上下移动
- **拖拽排序**：支持点位顺序调整

## 文件说明

- `autoclicker.py`：主程序文件
- `click_points.json`：自动保存的点位配置
- `autoclicker_log.txt`：程序运行日志
- `click_log.txt`：点击任务执行报告

## 配置说明

### 点位数据结构
```json
[
  {
    "x": 100,
    "y": 200,
    "name": "按钮1",
    "button": "left",
    "delay": 1.0
  }
]
```

### 主题配置
- 通过设置菜单切换浅色/深色主题
- 主题偏好自动保存
